A demo and stress test of the worker pool:

  • The output should count monotonically from 0 to 100.
  • The difference between the last input and last output should reach an upper bound smaller than 100.
  • The number of idle workers should remain at or near 0 until the end.
  • At the end, the number of idle workers should increase to the pool size.
This indicates that the pool controls the input flow, the outputs are delivered in order, the pool is fully utilized and no workers are leaked.
